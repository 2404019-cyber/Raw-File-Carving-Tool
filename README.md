# Raw-File-Carving-Tool
Description:

A C++ based Raw File Carving Tool that recovers deleted files by scanning raw disk sectors and identifying file signatures (magic numbers).

Features:
Recover JPEG files
Recover PNG files
Recover PDF files
Recover MP4 files
Recover ZIP files
Recover EXE files
Signature (Magic Number) based recovery
Object-Oriented Programming (OOP) design
Windows API support

Technologies Used:
C++
Object-Oriented Programming (OOP)
Windows API
File I/O

How to Run:
Open the project in Code::Blocks or Visual Studio.
Compile the source code.
Run the program as Administrator.
Select the target drive.
Select the file type to recover.
Recovered files will be saved in the Recovered folder.
